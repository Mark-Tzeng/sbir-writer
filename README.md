## ✨ 功能特色

- **三種申請階段**：完整支援 Phase 1、Phase 2、Phase 2+
- **逐步引導表單**：每個階段拆分為多個步驟，不漏填任何重要欄位
- **四種 AI 輔助模式**：
  - 🖊️ **草擬段落**：輸入想法，AI 自動轉換為計畫書語言
  - 💡 **填寫建議**：針對欄位提供撰寫重點與範例
  - ✨ **潤稿優化**：貼入已寫內容，AI 幫你精煉文字
  - 🔍 **審查視角**：模擬委員質疑，事先補強弱點
- **整合 50+ 知識庫**：涵蓋評分標準、成功案例、ROI 計算、產業指南等

## 🚀 使用方式

1. 開啟工具網頁
2. 在首頁輸入您的 **Anthropic API Key**（僅存於瀏覽器本機，不會外傳）
3. 選擇申請階段（Phase 1 / 2 / 2+）
4. 按步驟填寫，隨時呼叫右側 AI 面板輔助

> 💡 沒有 API Key？請前往 [console.anthropic.com](https://console.anthropic.com/) 申請（新帳號有免費額度）

## 🔑 如何取得 Anthropic API Key

### 步驟一：註冊帳號
1. 前往 [console.anthropic.com](https://console.anthropic.com/)
2. 點擊右上角 **「Sign Up」** 註冊帳號（支援 Google 登入）

### 步驟二：取得 API Key
1. 登入後，點擊左側選單 **「API Keys」**
2. 點擊 **「Create Key」**，輸入任意名稱（例如：`sbir-writer`）
3. 複製畫面上出現的 Key（格式為 `sk-ant-api03-...`）
   > ⚠️ Key 只會顯示一次，請立即複製儲存

### 步驟三：加值儲值
- 新帳號通常有少量免費額度，用完後需至 **「Billing」** 頁面加值
- 本工具每次 AI 請求約消耗 **$0.001–0.003 USD**（費用極低）

## ✏️ 在哪裡輸入 API Key

1. 開啟工具網頁後，**首頁最上方**會看到「🔑 設定 Anthropic API Key」區塊
2. 將複製的 Key 貼入輸入框
3. 點擊 **「儲存」** 按鈕
4. 看到「✅ API Key 已儲存於瀏覽器本機」即完成設定

> 💡 Key 儲存在您的瀏覽器中，**下次開啟不需重新輸入**。換瀏覽器或清除瀏覽器資料時才需重設。

### 隱私說明
- API Key **僅儲存於您的瀏覽器 localStorage**
- Key **不會**傳送至任何第三方伺服器
- API 費用由**使用者自行負擔**（直接呼叫 Anthropic，本工具不收取任何費用）
